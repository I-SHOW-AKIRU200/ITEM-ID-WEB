# 🛠️ ITEM-ID-WEB

A lightweight, fast, and responsive static website built to manage and retrieve Item IDs efficiently. 

## 🚀 Features
* **Instant Search:** Quickly find the IDs you need.
* **Clean UI:** Minimalist design for better focus.
* **Static & Fast:** Hosted directly via GitHub Pages with zero backend lag.
* **Mobile Friendly:** Fully responsive design.

## 🌐 Live Demo
Check out the live tool here: [https://i-show-akiru200.github.io/ITEM-ID-WEB/](https://i-show-akiru200.github.io/ITEM-ID-WEB/)

## 🛠️ Built With
* **HTML5** - Markup structure
* **CSS3** - Styling and layout
* **JavaScript** - Logic and interactivity

## 📂 Installation
If you want to run this project locally:

1. Clone the repository:
   ```bash
   git clone [https://github.com/I-SHOW-AKIRU200/ITEM-ID-WEB.git](https://github.com/I-SHOW-AKIRU200/ITEM-ID-WEB.git)

2. Navigate to the folder:
   ```bash
   cd ITEM-ID-WEB
3. Open index.html in your browser.
